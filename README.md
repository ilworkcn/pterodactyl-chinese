# Pterodactyl 翼龙面板本地化项目  

Pterodactyl® 是一个免费的开源游戏服务器管理面板，使用 PHP、React 和 Go 构建。  
此本地化项目仅负责官方稳定发行版汉化，不保证DEV分支使用汉化补丁的稳定性！  
所有 Releases 均经过构建测试，可放心使用！
